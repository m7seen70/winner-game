# Winner Web - نسخة ويب حقيقية كبداية

هذه النسخة **حقيقية من ناحية التشغيل عبر سيرفر ومتصفح** وفيها:
- تسجيل
- تسجيل دخول
- شرط اسم المستخدم:
  - 3 أحرف فأكثر
  - حروف إنجليزية وأرقام فقط
  - بدون مسافات
- لعبة زر **Realtime** عبر Socket.IO
- Top 20
- Announcement عام
- Recharge Codes
- Send Coins
- Admin Panel

## مهم
هذه ليست النسخة النهائية للإنتاج، لأن البيانات الآن **في الذاكرة**:
- إذا أعدت تشغيل السيرفر، يتم تصفير البيانات
- لا يوجد بريد حقيقي
- لا توجد قاعدة بيانات دائمة

لكنها **أفضل نقطة انطلاق حقيقية** لتجربة اللعبة على الويب.

## التشغيل على جهاز كمبيوتر
1. ثبّت Node.js
2. افتح Terminal داخل المجلد
3. نفّذ:
```bash
npm install
npm start
```
4. افتح:
```bash
http://localhost:3000
```
